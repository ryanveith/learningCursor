## Getting started

This folder is the local workspace for [learningCursor](https://github.com/ryanveith/learningCursor).

Use Cursor's chat and agent features here to try prompts, edits, and small experiments. Add notes or sample files as you go.

## Roman numeral converter

Interactive CLI to convert between Roman and Arabic numerals (integers 1–3999).

```bash
python roman_converter.py
```

Choose `1` for Roman → Arabic or `2` for Arabic → Roman, then enter a value when prompted.

Run unit tests (stdlib `unittest`, no extra dependencies):

```bash
python -m unittest
```

Verbose output: `python -m unittest -v`
